# PoultryFarmManagement
A Poultry Farm Management System built with Spring Boot and PHP, providing REST APIs for farm, flock, feed, batch, travel, company, and user management with secure authentication and automated data handling.

Features
🐔 Farm Management
🐣 Flock Management
📦 Batch Management
🌾 Feed Management
🚚 Travel/Transportation Management
🏢 Company Management
👤 User Authentication & Authorization
💾 Entry Management
📊 Farm Statistics & Reports
🔐 Login & User Profile APIs
